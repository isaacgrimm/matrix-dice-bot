# Matrix Dice Bot 🎲

Бот для сети Matrix, позволяющий бросать кубы прямо в чате через команды. Идеально подходит для настольных ролевых игр (НРИ), D&D, Pathfinder и других игр, где требуется бросание кубов.

## Возможности

- 🎲 Бросание различных типов кубов: d4, d6, d8, d10, d12, d20, d100
- 🔮 Поддержка кубов Fudge/Fate: `4dF`, `dFudge`
- ➕ Поддержка модификаторов: `!roll 2d20+5`, `!roll 3d6-2`
- 🔢 Несколько кубов за раз: `!roll 5d8`
- 🇷🇺 Поддержка русских команд: `!кинь`, `!брось`, `!кубы`
- ❓ Встроенная справка по командам
- 📊 Подробный вывод результатов с суммой

## Быстрый старт

### 1. Установка зависимостей

```bash
cd proj/matrix-dice-bot
pip install -r requirements.txt
```

### 2. Настройка конфигурации

Скопируйте шаблон конфигурации:

```bash
cp .env.example .env
```

Отредактируйте файл `.env`, указав ваши данные:

```bash
# Адрес Matrix сервера
MATRIX_HOMESERVER=https://matrix.org

# Ваши учётные данные бота
MATRIX_USERNAME=@yourbot:matrix.org
MATRIX_PASSWORD=your_password_here

# ID комнат (можно использовать алиас, несколько комнат через запятую)
MATRIX_ROOMS=!roomid1:matrix.org,!roomid2:matrix.org,#roomalias:matrix.org
# Для одной комнаты можно использовать MATRIX_ROOM:
# MATRIX_ROOM=!roomid:matrix.org

# Опционально: токен доступа (вместо пароля)
# MATRIX_ACCESS_TOKEN=your_access_token_here
```

### 3. Запуск бота

```bash
python bot.py
```

## Консольный режим для тестирования

Для быстрой проверки работы кубов без подключения к Matrix серверу используйте консольный режим:

```bash
python console/console.py
```

### Возможности консольного режима

- 🎲 Мгновенная проверка бросков кубов
- 🔧 Отладка форматов кубов без деплоя
- 📝 Поддержка всех форматов: `d20`, `2d6`, `4dF`, `dF` и т.д.
- 🇷🇺 Русские команды: `!кинь`, `!брось`

### Примеры использования

```
🎲 > !roll d20
🎲 d20 = [15] = 15

🎲 > !roll 4dF
🔮 4dF = [+ -  ] = 0

🎲 > !roll 2d6+3
🎲 2d6+3 = [4, 6]+3 = 13

🎲 > !кинь dF
🔮 dF = [+ + -] = 1

🎲 > quit
До свидания! 🎲
```

### Важные отличия

- `dF` → 4 куба Fudge (по умолчанию)
- `1dF` → 1 куб Fudge (явное указание)
- `d6` → 1 куб d6 (по умолчанию для стандартных кубов)

## Получение учётных данных Matrix

### Способ 1: Через пароль

1. Создайте нового пользователя в Matrix (через клиент Element или другой)
2. Используйте логин и пароль в `.env` файле

### Способ 2: Через токен доступа (рекомендуется)

1. Войдите в Matrix через веб-клиент (https://app.element.io)
2. Откройте настройки аккаунта → Sessions
3. Скопируйте Access Token
4. Используйте `MATRIX_ACCESS_TOKEN` вместо пароля

### Получение ID комнаты

В Element:
1. Откройте комнату
2. Нажмите на название комнаты → Settings
3. Скопируйте «Internal room ID» (вида `!abcdefghijk:matrix.org`)

Или используйте алиас комнаты (вида `#roomname:matrix.org`)

## Использование

### Команды бота

| Команда | Описание | Пример |
|---------|----------|--------|
| `!roll <кубы>` | Бросить кубы | `!roll d20` |
| `!d<стороны>` | Быстрый бросок | `!d6` |
| `!кинь <кубы>` | Русская команда | `!кинь 2d6` |
| `!брось <кубы>` | Русская команда | `!брось 3d8+2` |
| `!help` | Показать справку | `!help` |
| `!roll 4dF` | Кубы Fudge/Fate | `!roll 4dF` |

**Бот работает в нескольких комнатах одновременно!** Просто укажите несколько ID комнат в `MATRIX_ROOMS` через запятую.

### Формат записи кубов

#### Стандартные кубы: `NdS[+/-M]`
- `N` — количество кубов (необязательно, по умолчанию 1)
- `S` — количество сторон на кубе
- `M` — модификатор к результату (необязательно)

#### Кубы Fudge/Fate: `NdF[+/-M]`
- `N` — количество кубов Fudge (необязательно, по умолчанию 4)
- `F` — обозначение Fudge/Fate куба
- `M` — модификатор к результату (необязательно)
- Результат: от `-N` до `+N` (для 4dF: от -4 до +4)

### Примеры использования

```
!roll d20          # Один куб d20
!roll 2d6          # Два куба d6
!roll 3d8+2        # Три куба d8 + модификатор 2
!roll 4d10-1       # Четыре куба d10 - модификатор 1
!roll 1d100        # Один куб d100 (проценты)
!roll 4dF          # Четыре куба Fudge/Fate
!d6                # Быстрый бросок d6
!кинь d20          # Русская команда
!брось 2d6+3       # Русская команда с модификатором
!кинь 4dF          # Русская команда для Fudge
```

### Примеры вывода

```
!roll d20
🎲 d20 = 15

!roll 2d6+3
🎲 2d6+3 = [4, 5]+3 = 12

!roll 3d8
🎲 3d8 = [6, 2, 7] = 15

!roll 4dF
🔮 4dF = [+ -  ] = 0

!roll 4dF+2
🔮 4dF = [++ -]+2 = 2

!кинь d20
🎲 d20 = 8
```

## Поддерживаемые префиксы команд

Бот реагирует на сообщения, начинающиеся с:
- `!` (восклицательный знак) — основной префикс
- `/` (слэш) — альтернативный префикс
- `.` (точка) — альтернативный префикс

Примеры:
- `!roll d20`
- `/roll d20`
- `.roll d20`

## Работа с несколькими комнатами

Бот поддерживает работу сразу в нескольких комнатах Matrix. Для этого:

1. Укажите несколько ID комнат в переменной окружения `MATRIX_ROOMS`:
   ```bash
   MATRIX_ROOMS=!roomid1:matrix.org,!roomid2:matrix.org,#roomalias:matrix.org
   ```

2. Разделитель — запятая (пробелы вокруг запятой необязательны)

3. Бот автоматически присоединится ко всем указанным комнатам и будет отвечать на команды в каждой из них

4. Для совместимости со старыми конфигурациями можно использовать `MATRIX_ROOM` для одной комнаты

## Решение проблем

### Бот не подключается к серверу

1. Проверьте правильность `MATRIX_HOMESERVER`
2. Убедитесь, что учётные данные верны
3. Проверьте подключение к интернету

### Бот не видит команды

1. Убедитесь, что бот в нужной комнате
2. Проверьте префикс команды (`!`, `/`, `.`)
3. Включите DEBUG логирование для отладки

### Ошибка аутентификации

1. Проверьте логин/пароль или токен доступа
2. Для серверов кроме matrix.org укажите правильный адрес
3. Попробуйте получить новый токен доступа

### Fudge кубы не работают

1. Используйте формат `NdF` (например, `4dF`, `dF`)
2. Проверьте написание: `dF`, `dFudge`, `dFate`
3. Модификатор указывается после: `4dF+2`, `4dF-1`
